# 前端页面设计模板

这是个人多端同步记账软件的 Web 前端静态高保真原型，基于 `docs/ARCH.md`、`docs/PRD.md`、`docs/API.md` 和 `docs/DB.md` 设计。

## 打开方式

直接用浏览器打开：

```text
design/index.html
```

原型不依赖后端、不需要安装依赖、不需要构建命令。

## 页面范围

- 首页：本月收入、支出、结余、预算进度、近期账单、同步状态。
- 账单：列表、筛选、搜索、新增、编辑入口、删除确认演示。
- 统计：月度摘要、分类占比、年度趋势。
- 预算：总预算、分类预算、正常/预警/超限状态。
- 设置：账号信息、分类、账户、设备同步、备份恢复。

## 数据说明

`app.js` 内置 mock 数据，字段命名对齐接口文档：

- `transactions`
- `categories`
- `accounts`
- `budgets`
- `devices`
- `backupJobs`

金额使用整数分存储，页面展示时转换为人民币元。
